# Notice and provenance policy

## Bundled inspiration examples

Every inspiration example bundled with Kiro Collab Skill Kit is an original first-party example created for this project. The project does not bundle copied external skills, marketplace material, or content harvested from third-party repositories.

When an example is displayed in the application or documentation, it must identify its first-party origin. User-provided inspiration remains the user's responsibility and is used only for the requested local generation or test workflow unless the user explicitly chooses another supported storage mechanism in a future release.

## Policy for potential Kiro Hub-derived content

No Kiro Hub-derived content is bundled in this project. A Kiro Hub-derived item may be considered only after a documented audit confirms all of the following:

1. **License:** the applicable license permits the intended copying, modification, and redistribution under this project's distribution terms.
2. **Author:** the original author or rights holder is identified and the attribution requirements are recorded.
3. **Source:** the canonical source location and exact version or revision are recorded so the item can be traced and reviewed.
4. **Scope:** the proposed use contains no credentials, private data, proprietary material, or terms that conflict with the standalone project's local-first policy.
5. **Notice:** required attribution, license text, and modification notes are added before distribution.

If any audit step is incomplete or unclear, do not add the content. Link to the source for human review instead of copying it. This policy applies equally to examples, templates, prompts, test fixtures, and generated reference material.
